# Lead Scoring Case Study

## Project Overview
This project aims to build a machine learning model using Logistic Regression to predict the probability of lead conversion and assign lead scores to identify high-potential customers.

## Problem Statement
X Education wants to improve its lead conversion rate by identifying potential leads (hot leads) so that the sales team can focus on the most promising customers.

## Approach
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building using Logistic Regression
- Model Evaluation (Accuracy, ROC-AUC)
- Threshold Tuning
- Lead Scoring

## Results
The model successfully identifies high-potential leads and helps improve conversion efficiency by prioritizing leads based on their scores.
